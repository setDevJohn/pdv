# 2. Pesquisa de mercado — PDVs validados

*Referência de negócio. Útil para revisar planos e preços; não é usado no chat com o agente.*

| Sistema | Foco | Modelo de preço | Diferenciais |
|---|---|---|---|
| **MarketUP** | Pequeno comércio BR | Gratuito completo (PDV, estoque, NF-e/NFC-e, loja virtual); taxa simbólica após 1º ano | Único PDV genuinamente free no BR; suporte é o ponto fraco |
| **Bling** | Pequeno/médio comércio BR | Módulo PDV com taxa após ano 1; grátis para planos MarketUP+/Delivery+ | Forte em emissão fiscal e integração e-commerce |
| **GestãoClick** | Pequeno/médio comércio BR | Planos pagos (consultar) | PDV + estoque + financeiro nativamente integrados; TEF com Stone/Cielo/Rede/GetNet sem hardware extra |
| **vhsys** | Pequeno comércio com fluxo de caixa | A partir de ~R$100/mês, sem plano grátis | Robusto, porém caro para MEI de baixo volume |
| **Linx** | Médio/grande varejo | Sob consulta comercial | Integração ERP completa, foco em compliance fiscal |
| **PDV Smart** | Segmentado (varejo, food service, MEI, mobilidade) | A partir de R$99/mês | Landing pages por segmento; TEF + NFC-e ilimitada |
| **Loyverse** (referência internacional de UX) | Micro/pequeno varejo global | Grátis + Advanced Inventory a US$29/mês por loja | Interface muito elogiada por simplicidade; onboarding sem fricção |
| **Square** (referência internacional de UX) | Pequeno/médio varejo global | Grátis + planos pagos a partir de US$29/mês/local | Referência em design e fluxo de checkout |

**Faixa de mercado no Brasil:** de R$0 (versões gratuitas limitadas) a ~R$500/mês (soluções corporativas). Faixa saudável para pequeno/médio comércio: **R$99–R$199/mês**, incluindo NFC-e e suporte.
